# TripSplit Pro (Smart Group Expense Manager)

## Core Features

- Create trip
- Invite friends
- Add expenses (who paid, who joined)
- Multi-currency support
- Auto balance calculation
- “Who owes who” simplified settlement

### Senior-Level

1. Multi-Currency Conversion

- Store original currency
- Store exchange rate at time of expense
- Convert to base currency
- Lock exchange rate per transaction
- Now you deal with:
- Precision handling (decimal)
- Exchange rate consistency
- Historical rate storage

2. Smart Settlement Optimizer

Instead of: A pays B, B pays C, C pays A
Optimize to: A pays C only

3. Budget Tracking

- Set trip budget
- Track per-category spending
- Show over/under budget
- Forecast remaining days

4. Offline Mode

- Since trips sometimes no internet:
- Cache locally
- Sync when online
- Shows frontend maturity.
